# AI Image Gen App

- Node.js
- Express.js
- MongoDB
- React
- Tailwind CSS
- DALL-E model
- Cloudinary
- Vite

## Dependencies

```-bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

npm install file-saver
npm install react-router-dom
```

# Server

```-bash
npm init -y

"start": "nodemon index"

```

## Dependencies

```-bash
npm install cloudinary cors dotenv express mongoose nodemon openai

## add to package.json
"type": "module",
```
